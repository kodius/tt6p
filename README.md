# Time to six pack app

## Description

Get that ripped abs you always dreamed about.

## Installation

```
git clone git@github.com:kodius/tt6p.git
cd tt6p
bundle install
yarn install
bundle exec rails db:migrate
bundle exec rails db:seed
```

## Booting the app

```
foreman start
```

## Running tests

### Rails side

```
rails test
```

### JS side

```
yarn test
```
